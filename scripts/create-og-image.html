How to Create the OG Image

  1. Right-click on the preview below and select "Inspect" or "Inspect Element"
  2. In the DevTools, find the element with class og-image
  3. Right-click on that element and select "Capture node screenshot" (Chrome) or use a screenshot tool
  4. Save the image as og-image.jpg in the public/images/ folder
  5. Alternatively, use a design tool (Figma, Canva, Photoshop) to create a 1200x630px image
